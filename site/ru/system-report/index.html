system-report

JSON инвентаризация системы

Описание

Собирает информацию о системе в единый JSON документ: установленные пакеты, активные systemd юниты, открытые порты, точки монтирования, параметры ядра и информацию о железе.

Использование

system-report
system-report --output report.json
system-report --pretty

Секции вывода

СекцияОписание
packagesСписок установленных Nix пакетов с версиями
servicesВключённые и активные systemd юниты
portsСлушающие TCP/UDP порты
mountsТочки монтирования и использование
kernelВерсия и параметры ядра
hardwareCPU, память, диски
