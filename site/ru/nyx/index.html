nyx

Плагинный NixOS тулкит — ноль без плагинов, бесконечен с ними

Описание

nyx — единый командный интерфейс для всех инструментов nixos-pack. Сам по себе он ничего не делает — он обнаруживает установленные утилиты и предоставляет их как подкоманды. Установи любой пакет nixos-pack и nyx найдёт его автоматически.

Использование

nyx           # показать доступные команды
nyx health    # запустить nixos-health
nyx graph     # запустить nixos-module-graph
nyx top       # запустить nixos-top
nyx log -u nginx --follow
