nixos-module-graph

Анализ структуры импортов модулей NixOS

Описание

Обходит дерево imports = [...] NixOS модуля и строит граф зависимостей. Поддерживает HTML (d3.js), DOT, JSON и текстовый вывод.

Использование

nixos-module-graph --config /etc/nixos/configuration.nix
nixos-module-graph --config ./configuration.nix --output graph.html
nixos-module-graph --config ./configuration.nix --serve --port 8080
nixos-module-graph --config ./configuration.nix --text
nixos-module-graph --config ./configuration.nix --option services.nginx.enable
nixos-module-graph --flake .#my-server --output graph.html

Флаги

ФлагОписание
--configПуть к configuration.nix
--flakeFlake URI (например .#hostname)
--optionАнализ конкретной опции
--outputЗапись в файл (.html / .dot / .json)
--textASCII дерево в терминал
--serveЗапуск интерактивного d3.js сервера
--portПорт для --serve (по умолчанию 8080)
--max-depthЛимит глубины рекурсии импортов

Форматы вывода

  • HTML — интерактивный граф с d3.js
  • DOT — формат Graphviz
  • JSON — машиночитаемые узлы, рёбра, опции
  • Text — цветное ASCII дерево в терминале
