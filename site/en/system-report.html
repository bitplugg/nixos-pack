system-report

JSON system inventory

Description

Collects comprehensive system information into a single JSON document: installed packages, enabled systemd units, open ports, filesystem mounts, kernel parameters, and hardware info.

Usage

system-report
system-report --output report.json
system-report --pretty

Output sections

SectionDescription
packagesList of installed Nix packages with versions
servicesEnabled and active systemd units
portsListening TCP/UDP ports
mountsFilesystem mount points and usage
kernelKernel version and parameters
hardwareCPU, memory, disk info
