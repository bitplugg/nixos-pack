brrtfetch

Animated ASCII art from GIF

Description

Converts a GIF into smooth ANSI animations rendered in the terminal. Integrates with neofetch, fastfetch, or any sysinfo fetcher.

Usage

brrtfetch --gif dance.gif
brrtfetch --gif dance.gif --fetch fastfetch
brrtfetch --list-frames dance.gif

Options

FlagDescription
--gifPath to GIF file
--fetchSysinfo fetcher (neofetch, fastfetch)
--list-framesExtract and show all frames as text
--fpsAnimation speed in frames per second
