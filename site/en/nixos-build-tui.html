nixos-build-tui

Minimal TUI wrapper around nixos-rebuild

Description

Interactive terminal UI for nixos-rebuild. Provides action selection, build log streaming, rollback shortcut, and boot-entry management.

Usage

nixos-build-tui

Features

  • Select build action (switch, boot, test, dry-activate)
  • Live build log output
  • Quick rollback to previous generation
  • Boot entry management
