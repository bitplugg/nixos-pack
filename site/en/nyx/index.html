nyx

Plugin-based NixOS toolkit — zero alone, infinite with plugins

Description

nyx is a unified command interface for all nixos-pack tools. By itself it does nothing — it discovers installed tools and exposes them as subcommands. Install any nixos-pack package and nyx finds it automatically.

Usage

nyx           # show available commands
nyx health    # run nixos-health
nyx graph     # run nixos-module-graph
nyx top       # run nixos-top
nyx log -u nginx --follow
