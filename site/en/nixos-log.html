nixos-log

Journalctl browser with coloured output

Description

Wraps journalctl with syntax highlighting: errors and failures in red, warnings in yellow, service starts in green. Passes all flags through to journalctl.

Usage

nixos-log
nixos-log -u nginx --follow
nixos-log --boot -1 --priority err
nixos-log -p 5 -n 100
