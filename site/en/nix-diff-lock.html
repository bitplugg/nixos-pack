nix-diff-lock

Human-readable diff for flake.lock files

Description

Compares two flake.lock files and shows what inputs changed, what commits came in, and which dependencies were updated.

Usage

nix-diff-lock old.lock new.lock
nix-diff-lock --json before.lock after.lock

Output

  • Added inputs (green)
  • Removed inputs (red)
  • Updated inputs with old → new revisions
  • Summary of changed packages
