nixos-switch-diff

Show package diff between system generations

Description

Builds a candidate system configuration and diffs its store closure against the current generation. Shows what packages would be added, removed, or changed — without actually switching.

Usage

nixos-switch-diff

Output

  • + added — new packages in candidate
  • - removed — packages that would be removed
  • Build time summary
