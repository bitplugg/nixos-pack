rkn-block-checker

Multi-layer internet censorship diagnostics

Description

Diagnoses network blocks (RKN/TSPU) by testing connectivity at every layer: DNS resolution → TCP connection → TLS handshake → HTTP response. Identifies exactly where and how a resource is blocked.

Usage

rkn-block-checker example.com
rkn-block-checker --dns 1.1.1.1 example.com
rkn-block-checker --json example.com
rkn-block-checker --verbose example.com

Test layers

LayerWhat it checks
DNSResolution success, response time, spoofing detection
TCPTCP connection to port 80/443
TLSTLS handshake, certificate validity
HTTPHTTP response code and body analysis
