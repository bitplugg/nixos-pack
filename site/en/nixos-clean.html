nixos-clean

Nix store GC TUI with generation overview

Description

Shows a clean overview of all system generations and current nix store size, then runs garbage collection. Supports dry-run and full (historical) cleanup modes.

Usage

nixos-clean
nixos-clean --dry-run
nixos-clean --full

Options

FlagDescription
--dry-runShow how much would be freed without deleting
--fullCollect all historical garbage (including old generations)
--helpShow help
