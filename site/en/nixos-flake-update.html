nixos-flake-update

TUI for selectively updating flake inputs

Description

Interactive TUI that reads your flake.nix inputs, shows current locked revisions from flake.lock, and lets you pick which inputs to update — or update all at once.

Usage

nixos-flake-update

Controls

KeyAction
1-9Update specific input by number
aUpdate all inputs
qQuit
