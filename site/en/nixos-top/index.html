nixos-top

Real-time system monitor

Description

Live-updating terminal dashboard showing: load averages, memory usage, top systemd units by CPU, and recent journal errors. Updates every 2 seconds by default.

Usage

nixos-top
nixos-top -n 5

Options

FlagDescription
-nUpdate interval in seconds (default 2)
